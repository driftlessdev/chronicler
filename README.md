# Chronicler

Screenshot visual proof you killed that boss, dinged that level, or died embarrassingly. Chronicler will automatically capture the moments you want to relive later.

## Supported Events

- Leveling, with optional time to level
- Dungeon boss kills
- Raid boss kills
- Deaths this level
- Achievements
- Renown levels

## Future Potential Events
- Delve completions
- Loot wins

## Translators
Current support is only enUS for locales. If you are able, I'm accepting any and all translations.
